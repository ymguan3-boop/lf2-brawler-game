## 遊戲介紹

**官家大亂鬥**是一款以經典格鬥遊戲《小朋友齊打交 2》(LF2) 為靈感的橫版卷軸動作過關遊戲。全部程式碼集中在**單一 HTML 檔案**中，雙擊即可遊玩。

### 遊戲特色

- 🎮 **直式手機優化** — 9:10 畫面比例 + 虛擬搖桿 + 觸控按鈕
- 👊 **流暢戰鬥** — 出拳、踢擊、大絕招（冷卻10秒）、道具投擲
- 🆙 **升級系統** — 擊敗敵人獲得經驗值，升級可選擇能力卡片
- 🏆 **排行榜** — 分數自動記錄至 Supabase，TOP 10 排名
- 🤖 **多種敵人** — 小怪1/2/3/4、小王、小王2，各有不同 AI
- 👩 **招喚姑姑** — Z+X 組合鍵招喚 AI 夥伴輔助戰鬥
- 💥 **氣功波** — 升級出拳加速後，每次出拳發射氣功波

|------|------|
| ↑↓←→ | 四方向移動 |

## 開發階段與提示詞

### 階段 1：基礎骨架（2026年6月）

> **提示詞**：*「幫我做一個 LF2 風格的橫版格鬥遊戲，用單一 HTML 檔案，要有 Canvas 遊戲迴圈、鍵盤輸入、角色物理和碰撞檢測，畫布 960×540」*

- Canvas 遊戲迴圈（requestAnimationFrame）
- 鍵盤輸入系統（方向鍵 + Z/X/A/S/R）
- 角色物理（位置、速度、重力、邊界）
- AABB 碰撞檢測

### 階段 2：角色與動畫系統

> **提示詞**：*「更新主角精靈圖，檔名改為 主角動作_播放順序 格式」*
> **迭代修正**：*「朝向反了，請修正」→「所有主角圖片統一縮到 0.8x」→「特效產生點應在出拳平行位置與怪物接觸點」*

- 精靈圖異步載入 + fallback 繪圖
- 幀動畫播放器（startAnim / tickAnim）
- 主角動作：走路(4)、出拳(3)、踢擊(3)、大絕招(16)、受擊(1)、死亡(3)

### 階段 3：敵人系統

> **提示詞**：*「新增小怪2，圖片資料夾為小怪2動作包」→「新增小怪3」→「新增小王，踢飛攻擊判定 5 下，蓄力紅光，最後一擊踢飛主角」*
> **迭代修正**：*「小王攻擊行為改為隨機 attack/kick，每 10 次觸發踢飛」→「踢飛動畫 ×1.3 倍大」*

- 小怪1 ~ 小怪4 各有不同屬性與 AI

### 階段 4：戰鬥系統

## 技術架構

- **前端**：HTML5 Canvas + Vanilla JavaScript（單一 HTML 檔案）
- **音效**：Web Audio API（合成音效）
- **後端**：Supabase（排行榜資料庫）
- **動畫**：Sprite 幀動畫系統
- **物理**：AABB 碰撞 + 重力 + 反彈
- **鏡頭**：平滑跟隨 + 視差滾動背景
- **響應式**：CSS media query（`pointer:coarse`）+ 觸控事件

開啟瀏覽器前往：


直接雙擊 `index.html` 即可在瀏覽器中遊玩（無需任何伺服器或 build 工具）。

1. Fork 此倉庫
2. 修改 `index.html` 中的遊戲邏輯
3. 替換 `動作包/` 中的精靈圖
4. 推送至你的 GitHub Pages

## 部署方式

本專案使用 **GitHub Pages** 部署：

1. 推送至 GitHub 倉庫的 `main` 分支
2. 開啟倉庫 Settings → Pages
3. 選擇 **Deploy from a branch** → `main` → `/ (root)`
4. 等待幾分鐘，即可在 `https://<username>.github.io/lf2-brawler-game/` 遊玩

> 注意：本專案為純靜態網站，無需建置步驟，直接使用 root 目錄即可。
